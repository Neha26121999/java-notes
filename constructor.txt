                      Constructor

2)Does not have any return type, not even void.

3) The only modifiers are public, protected, default, private are applicable on constructor.

4)it executes automatically when we create object.there is no need to call them seprately.

5) constructors are used to initialize the object.

1.Default constructor: (no-arg constructor) - this constructor will create by compiler there is no role of jvm.
